# e-commerce-clone
It is an e-commerce website homepage

Technologies Used
Front-end: HTML, CSS, JavaScript
Back-end: Node.js, Express.js
Database: MongoDB
Deployment: Heroku
External APIs: Google Maps API, Payment Gateway API
